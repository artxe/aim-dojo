# Codex Instructions

`CLAUDE.md` files are the authoritative project instructions.

Before doing any work, read and follow the repository-root `CLAUDE.md`. Once the target files are known, also read and follow every `CLAUDE.md` from the repository root down to each target file's directory. If the target scope changes, load the newly applicable `CLAUDE.md` files before continuing.

Instructions closer to a target file override broader instructions when they conflict.
